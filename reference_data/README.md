## JRA-VAN データ仕様書について

JRA-VAN DataLabのデータ仕様書（PDF、XLSX等）は**再配布が禁止**されているため、このリポジトリには含まれていません。

### 仕様書の入手方法

JRA-VAN DataLab会員の方は、公式サイトからダウンロードできます：

1. **JRA-VAN DataLab 公式サイト**
   - https://jra-van.jp/dlb/

2. **開発者コミュニティ**
   - https://developer.jra-van.jp/

3. **データ仕様書 (JV-Data仕様書)**
   - 会員専用ページからダウンロード可能
   - PDF形式、XLSX形式で提供

### このディレクトリの使い方

開発時に参照用ドキュメントを配置したい場合は、このディレクトリに保存してください。

**注意**: `.gitignore` により、このディレクトリ内のファイルは Git にコミットされません。

```bash
reference_data/
├── README.md              # このファイル（Git管理対象）
├── JV-Data4901.pdf       # あなたがダウンロードした仕様書（Git管理対象外）
├── JV-Data4901.xlsx      # あなたがダウンロードした仕様書（Git管理対象外）
└── その他の参照資料        # Git管理対象外
```
